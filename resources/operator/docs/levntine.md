# 莱万汀

> "火焰，照亮黄昏！"

莱万汀来自罗德岛，以管理员直属干员的身份为终末地工业提供服务。

在华法琳的许可下，莱万汀曾在塔卫二各处独自游历，追寻她自己的目标。

她挥舞火焰，具有强大的独立作战能力，同时也是个待人真诚的人。

## 普通攻击：燃烬

**普通攻击**：对敌人至多造成5段攻击，造成灼燃伤害。作为主控干员时，重击会造成若干点失衡。

**下落攻击**：处于空中时使用普通攻击，会下落并攻击附近敌人，造成灼燃伤害。

**处决攻击**：附近有敌人处于失衡状态时使用普通攻击，将处决该敌人，造成大量灼燃伤害并恢复一定技力。

一轮总时长约为 3.8 秒，第五段为重击。重击回技 20 点, 造成 18 点失衡。

### 灼心

主控干员重击或处决命中时，莱万汀会吸收周围敌人的灼热附着。

每吸收一层灼热附着，自身获得一层熔火，最多叠加4层。

达到4层时，莱万汀造成的伤害无视敌人10/15/20点灼热抗性，持续20秒。

附近敌人被击败时，身上的灼热附着也会被吸收。

## 干员潜能

### 潜能1：熔火之心
战技焚灭的追加攻击的伤害倍率提升至原本的1.2倍，且命中时返还20点技力。

### 潜能2：追忆之旅
智识+20%，普通攻击伤害+15%。

### 潜能3：往事碎片
战技焚灭造成的燃烧持续时间+50%，且燃烧的伤害提升至原本的1.5倍。

### 潜能4：冰淇淋熔炉
终结技黄昏所需的终结技能量-15%。

### 潜能5：存在的证明
终结技黄昏的强化普通攻击的伤害倍率提升至原本的1.2倍，且终结技期间，莱万汀每击败一个敌人，终结技的持续时间+1秒。（最多+7秒）
